# LC 719. Find K-th Smallest Pair Distance
> tags: [BinarySearch] [SlidingWindow]

## Summary
1.  求「前 K 個 / 後 K 個」元素，可以使用 Binary Search。

<br>

## Solutions
### Solution #1
#### Techniques
- Binary Search
- Sliding Window

#### Resource Analysis
> N: |nums| <br>
> K: max(nums) - min(nums)
- Time: O(NlogN + NlogK)
- Space: O(1)

#### Note
```
1.  先將 nums 排序，當給定 threshold = k 後，
    就可以使用 sliding window 以 O(N)-time 算出距離 <= k 的 pair 有幾個
```
