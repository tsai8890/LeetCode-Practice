# LC 668. Kth Smallest Number in Multiplication Table
> tags: [BinarySearch]

## Summary
1.  求「前 K 個 / 後 K 個」元素，並且有單調性，可以使用 Binary Search。

<br>

## Solutions
### Solution #1
#### Techniques
- Binary Search

#### Resource Analysis
> N: n <br>
> M: m
- Time: O(Mlog(MxN))
- Space: O(1)

#### Note
```
1.  關鍵是給定 threshold = k 後，可以直接 traverse 1~m，
    計算出表上 <= k 的元素有幾個
```
