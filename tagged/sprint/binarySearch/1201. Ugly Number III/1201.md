# LC 1201. Ugly Number III
> tags: [BinarySearch]

## Summary
1.  求「前 K 個 / 後 K 個」元素，並且有單調性，可以使用 Binary Search。

<br>

## Solutions
### Solution #1
#### Techniques
- Binary Search

#### Resource Analysis
> K: 2000000000
- Time: O(logK)
- Space: O(1)

#### Note
```
1.  常用公式
    - gcd(a,b): 使用輾轉相除法
    - lcm(a,b) = a*b / gcd(a,b)
    - gcd(a,b,c) = gcd(gcd(a,b),c)
    - lcm(a,b,c) = lcm(lcm(a,b),c)
```
